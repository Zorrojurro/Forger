# Forger — AI Content Creator 🔥

> **Multiple specialized AI agents collaborate — Researcher, Writer, Editor, SEO Optimizer — to generate personalized content from a simple brief.**

## ✨ Features

- **Multi-Agent Pipeline** — 4 AI agents work sequentially, each refining the previous output
- **6 Content Types** — Blog posts, LinkedIn posts, X threads, video scripts, newsletters, podcasts
- **Real-Time Progress** — Watch agents work live with status updates
- **Export Options** — Copy, Markdown download, PDF export with Forger branding
- **Content Library** — Browse, search, and manage all generated content
- **Social Hub** — Schedule and publish directly to LinkedIn & X
- **Forger Watermark** — All content includes "Generated by Forger" branding
- **Smart Rate Limiting** — Circuit breaker + 11 req/min limit for free tier safety
- **Mock + AI Mode** — Works out-of-the-box with mock data, add a Gemini API key for real AI

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 16, TypeScript, Tailwind CSS |
| Backend | FastAPI, Python 3.13 |
| AI Agents | CrewAI-style agents with Gemini LLM |
| Workflow | Sequential pipeline (LangGraph-inspired) |
| Icons | Material Symbols Outlined |

## 🚀 Quick Start

### 1. Backend

```bash
cd backend
pip install -r requirements.txt
# Optional: copy .env.example to .env and add your GOOGLE_API_KEY
python -m uvicorn main:app --reload --port 8000
```

### 2. Frontend

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 🔑 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/health` | Health check |
| POST | `/api/generate` | Start content generation |
| GET | `/api/generate/{id}/status` | Pipeline progress |
| GET | `/api/generate/{id}/result` | Final content |
| GET | `/api/library` | Content library |
| POST | `/api/social/post` | Publish to social |
| GET | `/api/social/posts` | List social posts |

## 🧠 Agent Pipeline

```
Brief → 🔍 Researcher → ✍️ Writer → 📝 Editor → 🚀 SEO Optimizer → Final Output
```

Each agent has a specialized role:
- **Researcher** — Gathers facts, statistics, and trends
- **Writer** — Drafts content based on research, tone, and audience
- **Editor** — Refines grammar, flow, and readability
- **SEO Optimizer** — Adds keywords, meta descriptions, and hashtags
